## 關於 Line Bot - 迷因企鵝 (◉8◉) 
【迷因企鵝】是由我自製的一款 Line bot，一開始只有使用基本的自動回覆功能，完全沒有使用 API。因為想讓企鵝多學點才藝來娛樂大家，所以照著網路上各位前人大大們無私分享教學，試著（寫 code）讓他學習中。【迷因企鵝】的形象原型是來自日本人氣貼圖系列「めんトリ」中的『ぺんぐいん』（台灣翻：懶得"鳥"你，胖企鵝）。</br>
